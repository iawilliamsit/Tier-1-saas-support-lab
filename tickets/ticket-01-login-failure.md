# Ticket 01: Login Failure

## Ticket Summary
User cannot log into ClearDesk CRM.

## User Impact
User is unable to access the application and cannot complete assigned customer account work.

## Priority
Medium

## Category
Account Access

## Reported Symptoms
- User receives an invalid login message.
- User confirms the issue started today.
- User can access the internet and other applications.
- No other users are currently reporting the same issue.

## Questions Asked
- Are you using your assigned company email address?
- Did you recently change your password?
- Are you receiving an MFA prompt?
- Can you access other company applications?
- Are you using a saved browser password?

## Troubleshooting Steps
1. Confirmed the correct username format.
2. Verified the user had internet access.
3. Confirmed the issue was limited to ClearDesk CRM.
4. Asked the user to try a private browser window.
5. Confirmed no MFA prompt was appearing.
6. Guided the user through the password reset workflow.
7. Confirmed the user could log in after resetting the password.

## Result
Resolved at Tier 1.

## Customer Communication
Explained that the issue appeared to be related to account authentication. Walked the user through the password reset process, confirmed successful login, and advised the user to update any saved browser password.

## Resolution Code
Password Reset / Access Restored

## Escalation Needed?
No

## Internal Notes
This issue was resolved without Tier 2 escalation. If repeated login failures continue, check for account lockout, MFA enrollment status, or browser credential issues.

## Knowledge Base Opportunity
Create a short password reset and login troubleshooting guide.

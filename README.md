# Tier 1 SaaS Support and Escalation Lab

This project simulates a Tier 1 SaaS support workflow for a fictional CRM platform called ClearDesk CRM.

The goal is to demonstrate practical help desk and SaaS support skills, including ticket intake, user-impact assessment, troubleshooting, documentation, customer communication, resolution, escalation, and knowledge-base creation.

## Project Purpose

This lab was created to show how I approach front-line technical support work. It connects customer service experience, technical troubleshooting, documentation, and escalation judgment into a structured support workflow.

## Skills Demonstrated

- Ticket documentation
- User communication
- Account access troubleshooting
- Password reset workflow
- MFA and login issue triage
- SaaS application support
- Escalation notes
- Knowledge-base writing
- Remote support concepts
- SSH/Linux service-check concepts
- DNS and VPN troubleshooting basics

## Lab Scenario

ClearDesk CRM is a fictional SaaS platform used by employees to manage customers, accounts, files, appointments, and service activity.

The simulated support tickets in this project cover common Tier 1 issues such as login failures, password resets, account access requests, application navigation questions, device issues, Microsoft 365 issues, VPN/DNS problems, and escalation scenarios.
